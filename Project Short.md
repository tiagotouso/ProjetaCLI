# ProjetaCLI

> Este projeto oferece uma interface de linha de comando (CLI) para um gerenciamento de projetos ágil e descomplicado. Com ele, você pode facilmente planejar, executar e avaliar o ciclo de vida de seus projetos, mantendo um registro claro de informações, tarefas, marcos e correções.

* Setup Inicial
Inicia o repositório Git localmente
Cria o repositório remoto no GitHub
Configura o ambiente virtual Python
Cria a estrutura de pastas e arquivos
Instala dependências iniciais

* Modelagem de Dados
Cria o modelo para Project
Cria o modelo para Milestones
Cria o modelo para Actions
Cria o modelo para Issues
Cria o modelo para WorkLog
Desenvolve o módulo para salvar e carregar dados

* Lógica e Comandos da CLI
Criar a interface de linha de comando (CLI)
Gerenciar Project (add, list, show, set, update)
Gerenciar Milestones (add, list)
Gerenciar Actions (add, list, modify)
Gerenciar Issues (add, list, modify)
Gerenciar Worklog (add, list, modify, report)

* Qualidade e Documentação
Configura o ambiente de testes
Cria testes unitários para os modelos e a lógica principal
Cria testes de integração para os comandos da CLI
Mede a cobertura de testes do código
Configura uma ferramenta de documentação
Escreve o guia de uso na documentação
Cria um README.md completo com instalação e uso rápido

* Empacotamento e Distribuição
Configura o arquivo pyproject.toml para empacotamento
Gera o pacote distribuível
Cria o executável standalone
Cria a tag da versão
Publica a "Release" no GitHub com os executáveis anexados
